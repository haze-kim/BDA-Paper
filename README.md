# 단어 온톨로지를 적용한 서술형 강의평가 분석

> 학생 중심 강의평가 플랫폼 ‘에브리타임’의 서술형 강의평가 데이터를 활용하여, 학생들이 강의를 평가할 때 중요하게 인식하는 요소를 텍스트 마이닝과 단어 온톨로지 기반 분석으로 탐색한 프로젝트입니다.

## 프로젝트 소개

본 프로젝트는 정량적 강의평가 점수만으로는 파악하기 어려운 학생들의 실제 강의 경험과 평가 기준을 분석하기 위해 수행되었습니다.
에브리타임에 작성된 서술형 강의평가 텍스트를 수집하고, 강의평가 요소별 단어 온톨로지를 구축하여 리뷰 내용을 분류·분석했습니다.

특히 강의평가 텍스트에 나타나는 주요 평가 요소를 추출하고, 강의 유형이나 평가 양상에 따라 어떤 요소가 중요하게 작용하는지 확인하는 것을 목표로 합니다.

## 주요 분석 내용

* 에브리타임 서술형 강의평가 데이터 수집 및 전처리
* 강의평가 요소별 단어 온톨로지 구축
* 텍스트 마이닝 기반 핵심어 및 평가 요소 추출
* 강의평가 요소 간 관계 분석
* 의사결정나무 분석을 통한 강의평가 분류 기준 탐색
* 학생 관점에서의 강의 만족도 영향 요인 해석

## 연구 목적

기존의 강의평가는 대체로 정형화된 설문 문항과 점수 중심으로 이루어져 왔습니다.
그러나 학생들이 자유롭게 작성한 서술형 평가는 강의에 대한 구체적인 경험, 불만, 만족 요인을 더 풍부하게 담고 있습니다.

이 프로젝트는 이러한 비정형 텍스트 데이터를 분석하여 다음과 같은 질문에 답하고자 합니다.

1. 학생들은 강의를 평가할 때 어떤 요소를 주로 언급하는가?
2. 강의평가 요소는 강의 유형에 따라 어떻게 다르게 나타나는가?
3. 서술형 강의평가 텍스트를 통해 강의 만족도에 영향을 주는 요인을 파악할 수 있는가?

## 기대 효과

이 프로젝트는 학생 중심의 강의평가 데이터를 분석함으로써, 강의 개선에 필요한 실질적인 피드백을 구조화하는 데 목적이 있습니다.
또한 서술형 평가 데이터를 단순한 의견 모음이 아니라 분석 가능한 데이터로 전환함으로써, 교육 데이터 분석과 강의 질 개선 연구에 활용될 수 있습니다.
